# ATIKUL ISLAM RABBI — Official Portfolio Website

Official personal website and digital headquarters of Atikul Islam Rabbi, Social Media Security Specialist.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Static export for GitHub Pages

## Pages

- Home
- About
- Services
- Portfolio
- Blog (list + dynamic article routes)
- Resources
- Consultation
- Contact
- Privacy Policy
- Terms of Service
- Custom 404

## Development

```bash
pnpm install
pnpm lint
pnpm build
pnpm dev
```

## SEO/AEO

- Metadata per page
- Open Graph and Twitter cards
- JSON-LD schemas (Person, Organization, Service, FAQ, Article, Breadcrumb, Website)
- Sitemap and robots configuration

## Notes

- Content and positioning are aligned with documentation in `/docs`.
- Site is configured for static export (`output: "export"`) for GitHub Pages compatibility.
