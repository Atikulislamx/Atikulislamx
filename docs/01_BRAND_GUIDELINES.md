# Brand Overview

This website represents the official online presence of **Atikul Islam Rabbi**, a professional Social Media Security Specialist focused on helping individuals, creators, businesses, and organizations recover, secure, and protect their digital assets.

The website is not intended to function as a traditional freelancer portfolio. Instead, it serves as a premium digital headquarters that combines personal branding, professional credibility, educational resources, case studies, consultation services, and long-term trust building.

Every design decision must communicate professionalism, transparency, and confidence without relying on exaggerated claims or unnecessary visual effects.

Meet WCAG 2.2 AA where practical.

High color contrast.

Keyboard navigation support.

Visible focus states.

Proper semantic HTML.

Readable typography.

Responsive layouts across all devices.

# Brand Consistency Rules

Always use the official professional title:

Social Media Security Specialist

Never alternate between multiple titles across pages.

Maintain consistent colors, typography, spacing, and imagery throughout the website.

Every Google Flow image should follow the same visual language.

Every page should reinforce trust, professionalism, and clarity.

When adding future content, ensure it aligns with the brand mission and visual identity defined in this document.
