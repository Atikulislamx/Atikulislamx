# Brand Overview

This website represents the official online presence of **Atikul Islam Rabbi**, a professional Social Media Security Specialist focused on helping individuals, creators, businesses, and organizations recover, secure, and protect their digital assets.

The website is not intended to function as a traditional freelancer portfolio. Instead, it serves as a premium digital headquarters that combines personal branding, professional credibility, educational resources, case studies, consultation services, and long-term trust building.

Every design decision must communicate professionalism, transparency, and confidence without relying on exaggerated claims or unnecessary visual effects.

# 18. Component Design System

Every UI component should share the same design language.

## Buttons

Primary Button

• Solid electric blue background
• White text
• Rounded corners (12px)
• Medium shadow
• Smooth hover animation
• Scale: 1.02 on hover

Secondary Button

• Transparent background
• 1px border
• White text
• Border becomes accent color on hover

Ghost Button

• No border
• Transparent
• Underline on hover

Never use more than one accent color on the same button.

---

## Cards

Cards are one of the primary building blocks of the website.

Requirements

• Rounded corners (20px)
• Dark matte background
• Thin subtle border
• Soft blur background
• Very light shadow
• Comfortable padding

Cards should never look heavy.

The design should feel lightweight.

---

## Icons

Use Lucide Icons throughout the website.

Requirements

• Outline style
• Consistent stroke width
• No filled icons
• Size consistency

Avoid mixing icon libraries.

---

## Form Elements

Text Fields

• Rounded corners
• Dark background
• Blue focus ring
• Large padding

Dropdowns

• Minimal
• Rounded
• Accessible

Checkboxes

• Accent color only
